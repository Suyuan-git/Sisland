# vue-prop
## 安装
```
cnpm i
```
### 启动项目
```
npm run serve
```

### 打包项目
```
npm run build
```
### 本地服务器
```
serve dist
```
### 引用包、库
- [axios](https://www.kancloud.cn/yunye/axios/234845) 基于 promise 的 HTTP 库
- [vuex](https://vuex.vuejs.org/zh/) 专为 Vue.js 应用程序开发的状态管理模式
- [less](http://lesscss.cn/)  CSS 预处理语言
- [js-cookie](https://www.npmjs.com/package/js-cookie)
